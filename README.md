# DSA Lab

Lab exercises from my Data Structures & Algorithms course (CS 141), written in C. This repository will be updated throughout the semester, and will include all exercises by the end of April 2026.

## Built With

- [C](https://en.wikipedia.org/wiki/C_(programming_language)) - all exercises written in C

## Structure
```
dsa-lab/
├── lab1/
├── lab2/
├── lab3/
├── lab4/
└── ...
```

## Getting Started

### Prerequisites

Make sure you have a C compiler installed (e.g. GCC).

### Usage

1. Clone the repo
```
git clone https://github.com/meniarallouchi/dsa-lab
```
2. Navigate to any exercise folder
```
cd lab1
```
3. Compile and run
```
gcc exercise1.c -o exercise1
./exercise1
```
